# 資産リバランスツール

シンプルでわかりやすい資産配分計算ツールです。ブラウザで動作するため、環境構築不要。どのPCからでもすぐに使用できます。

## 特徴

- **インストール不要** - HTMLファイルをブラウザで開くだけ
- **オフライン対応** - ローカルで完全に動作
- **直感的UI** - シンプルで使いやすいインターフェース
- **リアルタイム計算** - 入力するとすぐに結果を表示
- **口数計算** - 投資信託の基準価額から必要な口数を自動計算
- **ビジュアル表示** - 現在の配分と目標配分をチャートで比較

## 機能

### 入力項目
- **現在の資産配分** - 各資産の現在の保有金額
- **目標資産配分** - 各資産の目標比率（%）
- **投資信託の基準価額** - 全世界株インデックスファンド、日経平均インデックスファンドの基準価額

### 資産クラス
- 全世界株インデックスファンド
- 日経平均インデックスファンド
- 日本株
- 米国株
- ビットコイン
- イーサリアム
- ソラナ
- 現金

### リバランス提案
提案結果には以下の情報が表示されます：
- 必要な買い増し/売却金額
- 基準価額が設定されている投資信託の場合、買い増し/売却に必要な口数
- 配分比率の変化（ポイント）
- 現在の配分と目標配分の差異をビジュアル表示

## 使い方

1. **ファイルを開く**
   - `asset-rebalancer.html` をダウンロード
   - ブラウザにドラッグ&ドロップするか、ダブルクリック

2. **現在の資産を入力**
   - 左側の「現在の資産配分」に各資産の金額を入力
   - リアルタイムで合計金額が表示されます

3. **目標配分を設定**
   - 右側の「目標資産配分」に各資産の目標比率を入力
   - 合計が100%になるように調整してください

4. **基準価額を入力（オプション）**
   - 投資信託を購入/売却予定の場合、基準価額を入力
   - 入力すると、購入/売却に必要な口数が表示されます

5. **計算実行**
   - 「リバランス提案を計算」ボタンをクリック
   - チャートとリバランス提案が表示されます

## 結果の見方

### リバランス提案

**例：売却の場合**
```
¥81,600
(2331口売却)
(-1.5ポイント)
```

**例：購入の場合**
```
¥29,600
(3947口購入)
(+0.6ポイント)
```

- **金額** - 買い増し/売却する金額
- **口数** - 基準価額が設定されている投資信託のみ表示
- **ポイント** - 配分比率の変化

## 技術仕様

### 口数の計算方法

投資信託の基準価額は「10,000口当たりの価格」を意味しています。

```
必要な口数 = (金額 × 10,000) ÷ 基準価額
```

**例：**
- 基準価額：100,000円（10,000口当たり）
- 買い増し金額：50,000円
- 必要な口数：(50,000 × 10,000) ÷ 100,000 = **5,000口**
